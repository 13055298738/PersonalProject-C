# 项目简介
## 项目组成
##### wordcount.cpp

## 如何运行？
#### 在DevC++下生成wordcount.exe后，在命令行下转入wordcount.exe所在的目录，input.txt必须在同一目录下,运行后自动生成output.txt文件

## 功能简介
####使用该程序进行对文件的字符数，单词数，行数，以及出现最多次的单词数进行统计。
